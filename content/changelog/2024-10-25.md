### Postgres extension updates

We added support for the following extensions to Postgres 17.

| Extension    | Version   |
| :----------- | :-------- |
| plv8         | 3.2.3     |
| HypoPG       | 1.4.1     |
| pgtap        | 1.3.3     |
| timescaledb  | 2.17.0    |

| pg_uuidv7    | 1.6.0     |
| wal2json     | 2.6       |
| pg_ivm       | 1.9       |
